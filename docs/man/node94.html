CLEAR: Clear the Terminal or Image Display

rawhtml282#
#command3370#

With no keywork or with the TEXT keyword, this command clears the command terminal screen.

With an X11 display window, the VEC keyword will clear the vector (overlay) list and redisplay the current images. The IMAGES keyword will clear the image list which is accessible with the #tex2html_wrap_inline3425# keys, resetting the list so that only the current image in included.
