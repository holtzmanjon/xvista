RPREP: Read a Record from a Batch Surface Photometry Prep-File

rawhtml456#
#command7104#

RPREP reads a record from a batch surface photometry preparation file opened using OPREP. These files contain the following information (one record per image):

  • <#7108#>SKY value (into VISTA variable SKY)<#7108#>
  • <#7109#>max. radius for photometry (into variable RMAX)<#7109#>
  • <#7110#>estimated HWHM for the PSF (into variable PSF)<#7110#>
  • <#7111#>contents of the AXES block (into AXES)<#7111#>
  • <#7112#>image header (ignored by this routine).<#7112#>

This information must already have been written into the prep-file using the WPREP. OPREP opens the file, which must have already been opened read-only using the `OPREP R' command

See Also: OPREP, WPREP, CLPREP
