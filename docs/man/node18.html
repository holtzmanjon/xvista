Extending Commands Past One Line

rawhtml72#

A VISTA command is ordinarily terminated by typing a RETURN, thus limiting the normal command length to one line of 80 characters. However, you can continue a command onto the next line by ending the current line of characters with the #tex2html_wrap_inline566# character. The last non-blank character must be a #tex2html_wrap_inline568# for your command to be extended. You will be prompted for the rest of the command with a colon on the next line. Example:


#hanging468#

There is a limit to the length of a command permitted even when using the #tex2html_wrap_inline572# character, and that limit is 500 characters.
