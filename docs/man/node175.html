EXTSPEC: Copy an Order of an Echelle Image to a Spectrum

rawhtml416#
#command6248#

This command copies a given order from an extracted echelle wavelength-calibrated image (where the row number of the Vista image corresponds to the order number of the wavelength-calibrated spectrum in that row) to a buffer. The order number is selected with the keyword `ORD='.
